# ShadowFox Contact Management System

Contact Management System built with Java as part
of ShadowFox Java Internship - Task 2 (Tier 1 Complete)

## Features

### Console Version
- Add, View, Update, Delete, Search contacts
- Phone & email validation
- Duplicate phone prevention
- Case-insensitive search

### GUI Version (Swing)
- TableView showing all contacts
- Double-click row to edit
- Search/filter contacts
- Confirmation dialog before delete
- Color coded buttons

## How to Run

### Console Version
1. Open in IntelliJ
2. Run `Main.java`

### GUI Version
1. Open in IntelliJ
2. Run `ContactAppGUI.java`

## Technologies Used
- Java 25
- Swing GUI
- ArrayList & HashSet
- Regex Validation

## Project Structure
src/
├── Main.java            → Console version
├── Contact.java         → Data model
├── ContactManager.java  → Business logic
└── ContactAppGUI.java   → Swing GUI
