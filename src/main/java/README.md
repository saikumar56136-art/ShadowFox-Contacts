# ShadowFox Contact Management System

A console-based Contact Management System built with Java
as part of ShadowFox Java Internship - Task 2.

## Features
- Add new contacts with validation
- View all contacts
- Search contacts (case-insensitive)
- Update contacts
- Delete contacts
- Duplicate phone number prevention
- Email and phone validation

## How to Run
1. Open project in IntelliJ IDEA
2. Run `Main.java`

## Project Structure
src/
├── Main.java            → Menu & user interface
├── Contact.java         → Data model
└── ContactManager.java  → CRUD logic
